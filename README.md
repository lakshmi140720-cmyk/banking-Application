# banking-Application


Banking Application is a simple digital banking system developed to perform basic banking operations easily and securely. This application allows users to login, check account balance, deposit money, withdraw money, make UPI transactions, and view transaction history. The main purpose of this project is to provide a user-friendly platform that helps customers manage their banking activities digitally. This project is developed using HTML, CSS, JavaScript, and backend technologies to understand the concepts of banking management systems and web application development.
